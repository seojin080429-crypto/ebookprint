F12 → Network → Copy URLs → 여기 → Console

ebook 페이지 → 콘솔 코드 생성기

이미지 URL 하나만 붙여넣으면 경로·접두사·자릿수를 자동 분석하고, 콘솔에 그대로 붙여넣을 코드를 만들어줍니다. 직접 구매·접근 권한이 있는 자료의 개인 소장용으로 사용하세요.

1이미지 URL 붙여넣기

아무 페이지 이미지 URL 하나면 충분합니다. 여러 줄을 붙여넣어도 첫 유효한 줄에서 패턴을 잡습니다.

2범위 입력
3방식 선택 후 코드 복사

모든 이미지를 먼저 받은 뒤 새 탭을 열고 인쇄창을 자동으로 띄웁니다. 로딩 끊김 없이 'PDF로 저장'만 누르면 됩니다.


    

  


  
  

    
사용 순서

    
    
      
  1. ebook 뷰어 탭에서 F12 → Console 탭 열기
    2. 
      
  2. 붙여넣기가 막혀 있으면 콘솔에 allow pasting 입력 후 Enter
    3. 
      
  3. 위에서 복사한 코드를 붙여넣고 Enter
    4. 
      
  4. (새 탭 방식) 새 탭에서 끝까지 한 번 스크롤해 이미지를 모두 로딩 → Ctrl+P → 'PDF로 저장'
    5.
