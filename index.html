F12 → Network → Copy URLs → 여기 → Console

ebook → PDF 도우미

두 가지 ebook 형태를 모두 지원합니다. 직접 구매·접근 권한이 있는 자료의 개인 소장용으로 사용하세요.

1뷰어 주소 전체 붙여넣기

브라우저 주소창의 URL을 통째로 복사해 붙여넣으세요. file= 뒤에 숨은 PDF 원본 주소를 자동으로 꺼내줍니다.

2PDF 원본 주소

아래 주소를 새 탭 주소창에 붙여넣어 여세요. PDF가 열리면 뷰어의 다운로드(↓) 버튼 또는 Ctrl+S로 저장하면 끝입니다.


      

      
    


    

      
PDF 뷰어형 사용 방법

      

        
A 주소에서 PDF 꺼내기

        
    
          
  1. ebook이 열린 탭의 주소창 URL을 전체 복사합니다. (주소창 클릭 → Ctrl+A → Ctrl+C)
    2. 
          
  2. 위 1단계 칸에 붙여넣으면 PDF 원본 주소가 자동으로 추출됩니다.
    3. 
          
  3. 새 탭에서 열기 버튼을 누르거나, 추출된 주소를 복사해 새 탭 주소창에 붙여넣습니다.
    4. 
          
  4. PDF가 열리면 다운로드(↓) 버튼 또는 Ctrl+S로 저장합니다.
    5. 
        

        
주소가 안 열리거나 로그인/인증이 걸리면(privw = 비공개일 수 있음) 아래 B 방법을 쓰세요.

      

      

        
B 안 열릴 때 — Network에서 잡기

        
    
          
  1. ebook 탭에서 F12 → Network 탭을 엽니다.
    2. 
          
  2. 필터에서 All 또는 Fetch/XHR을 누릅니다. (Img 아님)
    3. 
          
  3. Reload page(새로고침)를 한 번 누릅니다.
    4. 
          
  4. 목록에서 Type이 pdf이거나 크기(Size)가 큰 항목을 찾습니다.
    5. 
          
  5. 우클릭 → Open in new tab → 열린 PDF를 저장합니다.
    6.

모든 이미지를 먼저 받은 뒤 새 탭을 열고 인쇄창을 자동으로 띄웁니다. 로딩 끊김 없이 'PDF로 저장'만 누르면 됩니다.

    

      
      

    

  


  
  

    
전체 사용 방법 (처음부터 끝까지)


    

      
A F12로 이미지 링크 복사하기

      
    
        
  1. ebook 뷰어가 열린 탭에서 F12를 눌러 개발자 도구를 엽니다. (안 열리면 Ctrl+Shift+I)
    2. 
        
  2. 상단 탭 중 Network(네트워크)를 클릭합니다.
    3. 
        
  3. 그 아래 줄에서 Img 버튼을 클릭합니다. (이미지 요청만 걸러서 보여줍니다)
    4. 
        
  4. ebook에서 페이지를 두세 장 넘깁니다. 그러면 목록에 ...0005.jpg 같은 이미지 요청들이 줄줄이 뜹니다.
    5. 
        
  5. 목록의 아무 항목이나 우클릭 → Copy → Copy URL을 누릅니다. (이미지 URL 하나가 복사됩니다)
    6. 
      

      
목록이 비어 있으면 페이지를 더 넘기거나, F12를 연 채로 F5로 새로고침한 뒤 다시 넘겨보세요.

    


    

      
B 위에서 코드 만들기

      
    
        
  1. 1단계 칸에 방금 복사한 URL을 붙여넣습니다. → 경로·접두사 등이 자동으로 분석됩니다.
    2. 
        
  2. 2단계에서 끝 페이지를 책의 총 페이지 수로 맞춥니다. (시작은 보통 1)
    3. 
        
  3. 3단계에서 '새 탭 + 인쇄'를 선택하고 복사 버튼을 누릅니다.
    4. 
      

    


    

      
C 콘솔에 붙여넣어 PDF 만들기

      
    
        
  1. F12 개발자 도구에서 이번엔 Console(콘솔) 탭으로 이동합니다.
    2. 
        
  2. 처음이면 붙여넣기가 막혀 있습니다. 콘솔에 allow pasting 을 직접 타이핑하고 Enter. (한 번만 하면 됩니다)
    3. 
        
  3. 복사한 코드를 붙여넣고 Enter를 칩니다.
    4. 
        
  4. 콘솔에 [10/249], [20/249]... 진행 표시가 뜹니다. 끝까지 기다립니다. (1~2분)
    5. 
        
  5. 다 받으면 새 탭이 열리고 인쇄창이 자동으로 뜹니다.
    6. 
        
  6. 인쇄창에서 '대상(프린터)'을 'PDF로 저장'으로 바꾸고 저장을 누릅니다. 끝!
    7. 
      

      
팝업이 차단됐다는 메시지가 뜨면, 주소창 오른쪽의 팝업 차단 아이콘을 눌러 '허용'한 뒤 코드를 다시 실행하세요.
